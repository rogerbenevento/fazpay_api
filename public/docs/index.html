Introduction

Base URL: http://localhost:8880

Essa documentação lista os endpoints da api de teste da FazPay.

Essa API abrange a gestão de usuários e produtos.

Authenticating requests

This API is not authenticated.

Endpoints

Lista de produtos.

Lista os produtos de forma paginada, com filtro por nome, com o parametro via queryString:

  • search: string
Example request:
curl --request GET \
    --get "http://localhost:8880/api/products" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8880/api/products"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

    method: "GET",
    headers,
}).then(response => response.json());

curl --request POST \
    "http://localhost:8880/api/products" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
